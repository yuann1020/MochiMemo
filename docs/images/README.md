# Screenshot Images

This folder holds all screenshots used in the main README.md.

## Expected files

| Filename | Screen | Notes |
|---|---|---|
| `hero.png` | Full hero banner | Wide crop or collage showing the app's key screens |
| `home.png` | Home tab | Monthly overview, category donut, recent expenses visible |
| `add-expense-voice.png` | Add Expense — Voice tab | Microphone orb visible, idle or recording state |
| `add-expense-type.png` | Add Expense — Type tab | Text input visible with an example typed in |
| `review-expense.png` | Review Expense | At least one extracted expense card visible |
| `history.png` | History tab | Expenses grouped by date (Today / Yesterday labels visible) |
| `insights.png` | Insights tab | Category donut + AI Insights section visible |
| `budget-alert.png` | Budget Alert | Budget progress bar and remaining balance visible |
| `profile.png` | Profile tab | Display name and monthly budget fields visible |

## Notes

- Use iPhone screenshots for consistency (375 pt wide recommended).
- Avoid real personal data — use sample expenses like `Lunch RM18`, `Bubble tea RM12`.
- Crop to the safe area — remove the iOS status bar if it shows personal info.
- Save as PNG for best quality.

See [../SCREENSHOTS.md](../SCREENSHOTS.md) for the full capture guide.
