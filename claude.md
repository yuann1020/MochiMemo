# MochiMemo

Voice-first AI spending companion for iPhone.

Tech stack:
- Expo React Native
- TypeScript
- Supabase
- OpenAI APIs

Core flow:
1. User records spending by voice
2. AI transcribes the audio
3. AI extracts expenses into structured JSON
4. User confirms or edits
5. Expenses are saved to Supabase
6. App generates spending insights

Design:
- cute
- modern
- Gen Z
- mascot-driven
- smooth animations
- not corporate fintech

Rules:
- keep code clean
- avoid overengineering
- explain changes clearly
- build step by step
